# Blipyy v2.5.0

Released: April 23, 2026

Blipyy v2.5.0 rolls the latest self-hosted `develop` changes into `main`, with stronger account and data protections, new process-focused achievements, improved notification flows, and broader CSV import support including Tradovate paired exports.

## Highlights

- Hardened authentication, token handling, export safety, and account-linking flows across the backend.
- Added the first batch of process-focused achievements and improved how import achievements surface in notifications and history.
- Expanded CSV import resilience with better diagnostics, parser fallbacks, and support for Tradovate paired trade exports.

## Fixes And Improvements

- Corrected calendar and execution-level P&L alignment in day-detail and overview experiences.
- Improved notification dropdown behavior, unread-state handling, and related import UX.
- Refreshed key home-page and app UI surfaces for better accessibility and responsiveness.
- Updated dependencies and security overrides included in the shipped app.

## Upgrade Notes

- Run the latest backend migrations before deploying this release. This release adds migrations `173_dedupe_subscriptions_enforce_single_row.sql`, `174_create_oauth_pending_states.sql`, and `175_add_process_achievements_batch1.sql`.
- This release is backward-compatible and does not introduce intended breaking changes.
