# Blipyy v2.5.2

Released: April 27, 2026

Blipyy v2.5.2 is a patch release focused on CSV import correctness, preserving execution-level trade data during edits, and clearing current dependency vulnerability findings.

## Fixes

- Fixed generic CSV imports for completed-trade exports that include opening and closing columns, day-first timestamps, fractional seconds, lot-based quantities, and net P&L values.
- Fixed Tradovate execution preservation when editing trades so execution-level data is retained instead of being collapsed or lost.
- Updated vulnerable backend and frontend dependencies flagged by Snyk, including `axios`, `uuid`, `hyperid`, and `postcss`.

## Upgrade Notes

- This release is backward-compatible and does not introduce intended breaking changes.
